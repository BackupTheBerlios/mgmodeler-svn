Installer MG Modeler



MGModeler utilise automake et autotools.

Installation à partir des sources:

Aprés avoir décompressé l'archive il faut ce placer dans le répertoire et taper ./configure puis make et enfin make install. Il est possible de changer le répertoire de destination pour les binaires grace à l'option --prefix=path de configure.

Installation à partir du svn:

Pour récupérer les sources il suffit de taper svn checkout svn://svn.berlios.de/mgmodeler. Pour compiler ce placer dans mgmodeler puis taper ./bootstrap.sh, ensuite taper les mêmes commandes que pour l'installation depuis l'archive (./configure, ...).

Lancer MG Modeler



Pour cela il suffit de ce placer dans le répertoire $prefix/bin et de taper ./MGModeler.

Interface de MG Modeler



La fonction de chaque boutton de la barre d'outils est affichée si on laisse la souris dessus par l'intermédiaire d'une étiquette. Lors de l'édition d'objets le boutton du milieu de la souris sert à déplacer un point (points en rouge) sauf pour les cercles, dans la mesure ou il suffit de cliquer avec le bouton gauche pour redéssiner ailleur le cercle. Le boutton de droite sert à supprimer un point et shift + boutton gauche permet de rajouter un points entre deux autres. Dans la vue 3d la molète sert pour le zoom et le boutton gauche pour tourner autour du modèle.
